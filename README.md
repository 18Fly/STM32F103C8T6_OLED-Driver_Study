# <p align="center">项目说明</p>

- 这次上传的项目终于对了🤭
- 本次学习的是`SSD1306`屏幕驱动芯片的使用
- 通过`STM32F103C8T6`与`SSD1306`使用`I2C`进行通讯
- 分别实现了如何`打印字符、打印字符串、打印BitMap图片、打印汉字字面量(UTF-8)`
- 项目开发环境: `VSCode、EIDE(VSCode扩展插件)、Keil uVision5`
- 烧录工具: `ST-Link V2`
